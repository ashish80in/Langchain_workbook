# Day 5 - Text Splitters
This module demonstrates how to split text and documents into manageable chunks using LangChain.
## Files
- `01_basic_splitter.py` – Simple character-based splitting
- `02_recursive_splitter.py` – Recursive splitting with multiple separators
- `03_chunk_overlap.py` – Demonstrates chunk overlap
- `04_split_pdf.py` – Splitting text from PDFs

## Topics Covered
- CharacterTextSplitter
- RecursiveCharacterTextSplitter
- Chunk Size
- Chunk Overlap
- split_text()
- split_documents()
